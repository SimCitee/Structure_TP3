Structure_TP3
=============

- Au demarrage du programme, les donnees ne sont pas chargees (elles sont contenus dans les fichiers binaires etudiants.ser, cours.ser et inscription.ser). Il faut les charger à partir du menu principal (option 7).

- Si par megarde les donnees sont sauvegardees avant de les chargees, les fichiers de donnees se retrouveront vide. Ainsi, les fichiers originaux de donnees se trouvent dans le dossier "backup" (il suffit a ce moment de les copiers a la racine du projet).
